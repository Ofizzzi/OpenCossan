Stochastic ranking

for constrained evolutionary optimization

Thomas Philip Runarsson and Xin Yao, Stochastic Ranking for Constrained Evolutionary Optimization, IEEE Transactions on Evolutionary Computation, Vol. 4, No. 3, September 2000.

The MATLAB source code will allow you to re-run all experiments presented in the paper. The current release is 1.1a.

MATLAB file listing

  • Script files:
    run.m:
    run this script file to perform experiments.
    mktable.m
    when run.m has finished use mktable.m to make a LaTeX table of results.
  • Mex files: (Updated for MATLAB 6.0)
    The mex files must be compiled from within MATLAB using the commands:
    >> mex srsort.c
    >> mex arithx.c
    >> mex g12.c
     (this package includes pre-compiled MEX binaries in the directory \sres\binaries\)
    srsort.c:
    the stochastic ranking procedure.
    arithx.c:
    global intermediate recombination for the "mean step sizes".
    g12.c:
    test function G12.
  • Function:
    sres.m:
    actual ES using stochastic ranking.
  • Test function suite:
    g**.m:
    where ** denotes the test function number.
  • Miscellaneous:
    Joines and Houck experiment was added in the revised version for comparison.
    jhes.m:
    the same ES but using Joines and Houck's dynamic penalty method.
    runjh.m:
    run this script file to do the Joines and Houck experiment.

Use HELP in MATLAB on any of these files or TYPE them for more information.
