COSSAN-X Engine Toolbox: Features

The main features of COSSAN-X includes

Uncertenty Quantification

  • Stochastic Finete Element
  • Monte Carlo method and Quasi Monte Carlo methods (Latin Hypercube Sampling, Halton Sampling, Sobol Sampling).

Reliability Analysis

  • Monte Carlo method and Quasi Monte Carlo methods
  • Advanced Monte Carlo methods (Line Sampling, Importance Sampling, SubSet Simulation)
  • Approximate method based on the Design Point
  • Fault Tree and Cut-Set identification
  • System Reliability Analysis

Sensitivity Analysis

  • Local Sensitivity Analysis (based on finite diffence method and Monte Carlo method)
  • Global Sensitivity Analysis (Random Balance Design, Saltelli's method, Upper Bounds)

Optimization

  • Genetic Algorithms
  • Simulated Annealing
  • Cross Entropy
  • Evolution Strategy
  • COBYLA
  • SIMPLEX
  • BFGS
  • MiniMax
  • Reliability Based Optimization

Meta Modelling

  • ModeBased
  • Neural Networks
  • Polynomial Chaos
  • Response Surface
