TUTORIALPARAMETER

This tutorial shows the basics on how to define an object of the class Parameter

See Also: http://mechanik.uibk.ac.at/wiki/index.php/@Input

Contents

  • Create empty object
  • Create Parameter object
  • Access to the value

Create empty object

Xpar1     = Parameter;

% show summary of the object
display(Xpar1)

Parameter Object  -  Description: 
* Number of elements: 0

Create Parameter object

Xpar2   = Parameter('Sdescription','My Parameter','value',2);

% show summary of the object
display(Xpar2)

Parameter Object  -  Description: My Parameter
* Number of elements: 1

Access to the value

Val = Xpar2.value;

Nelement = Xpar2.Nelements;
